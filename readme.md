# HybridiSpeksi

1. Env setup

2. Deployment


### Env setup

Clone the project

    $ cd hybridispeksi
    $ npm install

Start backend-server:

    $ npm start

Start client server:

    $ npm run dev

### Deployment

Build project. Built files will be in /dist -folder.

    $ npm run build-prod 
    
To run in prod mode, npm start with 

    NODE_ENV=production
